# SpringBoot
主要目标：  
使用application.properties文件进行多环境配置（test, dev, production）  
使用spring security管理用户登录 (JWT)  
使用redis管理session和缓存  
使用spring-data-jpa简化数据层访问（MySQL）  
使用Async实现异步调用（email发送）  
配合vue或者react开发前端（桌面，平板，手机）  
使用七牛云CDN
